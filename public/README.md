# 🛍️ Vue Product Showcase

Aplicación SPA desarrollada con Vue.js que permite visualizar un catálogo de productos dinámico, con filtrado por categorías y diseño moderno.

---

## 🚀 Tecnologías utilizadas

* Vue.js
* Vuex
* Axios
* Vuetify (UI)
* Jest (testing)
* Cypress (E2E)

---

## 📦 Funcionalidades

✔ Visualización de productos desde API
✔ Filtrado por categorías
✔ Gestión de estado con Vuex
✔ Diseño responsive
✔ Pruebas unitarias y end-to-end

---

## 🧠 Decisiones técnicas

Se utilizó Vuex para centralizar el estado y evitar el paso excesivo de props entre componentes, mejorando la escalabilidad.

Vuetify fue seleccionado para lograr una interfaz moderna y consistente sin necesidad de diseñar desde cero.

---

## ▶️ Instalación

```bash

npm run serve
```

---

## 🧪 Pruebas
